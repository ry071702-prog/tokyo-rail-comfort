## 著作物利用の注意 (減点・失格リスクの回避)

- **Webサイトの画面キャプチャ**: 利用規約に準拠し、可能なら出典明記  有料記事の画面引用は避ける
- **地図 (Google Map等)**: 動画・画像キャプチャは可だが **Googleのロゴ・帰属表示を隠したり改変しない**  難しければ規約上問題ないサイトを使う
  - → Phase1 で地図を入れるなら MapLibre + OpenStreetMap (帰属表示付き) が安全
- **音源 (BGM)**: 許諾範囲の見極めが難しいので**使用を避ける**  静かな場所で録音し背後の音に注意
- **音声合成ナレーション**: 合理的理由 (発声が難しい等) がある場合のみ  時間短縮・読み間違い防止目的の利用は避ける
- **AI生成の加工素材**: 元素材の利用規約の範囲内で  不明なら使わない
  - → 本プロジェクトの方針 (架空IP・キャラを作らない / 装飾は抽象幾何・タイポ・ブランド色) と一致
